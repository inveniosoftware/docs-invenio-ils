# CLI reference

The following documents are a reference guide to the commands provided by the Invenio-CLI tool.

**Note:** The guide describes the functionality of invenio-cli that is used for working with InvenioILS instances.

**Intended audience**

This guide is intended for system administrators and developers of InvenioILS.

## Command Categories

- [General Commands](general-commands.md) - Installation, initialization, and system management
- [Services Commands](services-commands.md) - Docker services management for development
- [Users & Authentication](users-authentication.md) - Managing users, roles, tokens, and access permissions
